diffur.py - решение СДУ при помощи файла входных 
параметров системы  и файла с СДУ
sdy_form.py - формирование файла СДУ по файлу 
со стехиометрической матрицей
zakon_soxr.py - проверка закона сохранения количества по стадиям
при помощи файла с атомарно-молекулярной и стехиометрической матрицами

текстовые файлы:
- C0.txt: начальные данные по концентрациям
- E.txt: энергии активации
- k0.txt: предэкспоненциальные множители
- T.txt: температуры
- sdy.txt: сформированный в sdy_form.py файл с СДУ
- stex_matrix: стехиометрическая матрица
- stex_brom: стехиометрическая матрица для брома
- atm.txt: атомарно-молекулярная матрица для брома
